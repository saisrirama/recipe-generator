# Monorepo Documentation

This is a monorepo setup for the Recipe Generator project. It contains several packages that can be developed in isolation and published separately. This structure helps in managing dependencies and ensuring that related code is kept together.

## Packages

- `package-1`: Description of package 1
- `package-2`: Description of package 2

## Getting Started

To get started with the monorepo:
1. Install the dependencies:
   ```bash
   npm install
   ```
2. Run your desired package:
   ```bash
   npm run <package-name>
   ```

## Development

Use the following commands for development purposes:
- `npm run build`: Build the packages.
- `npm run test`: Run the tests.

## Conclusion

This documentation provides a basic overview of the monorepo structure. Make sure to refer to individual package documentation for more details.
